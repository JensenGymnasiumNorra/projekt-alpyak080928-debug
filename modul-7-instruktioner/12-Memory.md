# Projekt 12: Memory

## Målsättning
Skapa spelet Memory där spelaren vänder två kort per tur och försöker hitta alla par.

## Ska-krav
- Skapa ett bräde med dolda kort i par.
- Slumpa placeringen av korten inför varje spel.
- Låt spelaren välja två positioner per tur.
- Matchande kort ska ligga kvar öppna, annars döljas igen.
- Spelet avslutas när alla par är hittade.

## Kan-krav
- Lägg till stöd för två spelare med poängräkning.
- Lägg till svårighetsgrader med olika brädstorlek.
- Validera att samma kort inte kan väljas två gånger i samma tur.
- Lägg till tidsmätning och highscore.

## Förslag på testfall
- Två lika kort ger poäng och blir öppna.
- Två olika kort döljs igen.
- Spelet avslutas korrekt när alla par är hittade.
