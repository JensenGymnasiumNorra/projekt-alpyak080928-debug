# Instruktioner projekt

## Arbetsprocess
1. Välj ett projekt från mappen `modul-7-instruktioner` som du vill arbeta med.
2. Läs igenom instruktionerna för uppgiften.
3. Skapa en txt-fil med samma namn som projektet
4. Skriv psuedokod eller planera din lösning i txt-filen.
5. Implementera din lösning i en Python-fil med samma namn som projektet.
6. Testa ditt program noggrant och låt en klasskamrat testa.

## Tänk på att
1. Kommentera din kod tydligt så att andra kan förstå din tankeprocess.
2. Använd meningsfulla variabelnamn som beskriver deras syfte.
